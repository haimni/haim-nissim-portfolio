# FPGA & Hardware Validation Engineer Portfolio

## מטרת הפרויקט

מטרת הפרויקט היא לבנות אתר תיק עבודות אישי ומקצועי המציג את הניסיון, היכולות והפרויקטים שלי בתחומי:

- FPGA ו־RTL Design
- Hardware Validation
- Embedded Systems
- System Integration
- Python ואוטומציה
- PCB Design

האתר מיועד לשמש ככרטיס ביקור מקצועי עבור מגייסים, מנהלים טכניים, מהנדסים ושותפים פוטנציאליים.

העיצוב מתמקד במראה טכנולוגי, נקי ומינימליסטי, עם היררכיה ברורה בין אזורי התוכן ועם דגש על קריאות, נגישות והתאמה למסכים שונים.

## מבנה האתר המתוכנן

האתר יכלול את האזורים הבאים:

1. **Header וניווט**
   - שם מלא
   - קישורים לאזור הפרויקטים, הניסיון, אודות ויצירת קשר

2. **Hero Section**
   - כותרת מקצועית
   - תיאור קצר של תחום ההתמחות
   - כפתורים לצפייה בפרויקטים ולהורדת קורות החיים
   - תמונת פרופיל או מונוגרמה

3. **Skills / Technologies**
   - הצגת הטכנולוגיות והכלים המרכזיים
   - לדוגמה: VHDL, Vivado, SystemVerilog, Python, STM32, KiCad ו־PCB Design

4. **Featured Projects**
   - הצגת מספר פרויקטים מרכזיים
   - לכל פרויקט יוצגו שם, תיאור, טכנולוגיות וקישור למידע נוסף

5. **Experience**
   - ניסיון מקצועי
   - תפקידים, חברות ושנים

6. **Contact**
   - כתובת דוא״ל
   - LinkedIn
   - GitHub
